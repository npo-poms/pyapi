# pyapi
Python scripts to talk to the NPO api

#How to install
```
pip3 install git+git://github.com/npo-poms/pyapi.git
```

Then you can do:
```bash
michiel@belono:~$ media_get.py POMS_NTR_388772
No configuration file found. Now creating.
Your NPO api key?:
Your NPO api secret?:
Your NPO api origin?:
```
